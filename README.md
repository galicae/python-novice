# python-novice

Gentle introduction to Python for biologists with a focus on algorithmic thought and
problem-solving. For a rendered version of this material, please refer to the [GitHub
pages][https://galicae.github.io/python-novice/].

This lesson is built with sandpaper, following [the lesson template
documentation][https://carpentries.github.io/sandpaper-docs/] for formatting and building lesson
material. It is based on the [novice Python
course][https://github.com/swcarpentry/python-novice-gapminder] developed and used by The
